# 📅 60天全栈前端复习计划

## 第 1 周 - HTML & CSS 基础夯实

### Day 1: HTML5 语义化与最佳实践

- 语义化标签（header、nav、main、article、section、aside、footer）
- 表单元素与验证（input types、required、pattern）
- meta 标签与 SEO 优化
- 可访问性（ARIA、alt、role）
- 实践：创建一个语义化的博客文章页面

### Day 2: CSS 基础深度复习

- 选择器优先级计算（!important、内联、ID、类、标签）
- 盒模型详解（content-box vs border-box）
- 定位机制（static、relative、absolute、fixed、sticky）
- 层叠上下文与 z-index
- 实践：实现各种定位布局场景

### Day 3: Flex 布局精通

- 容器属性（flex-direction、justify-content、align-items、flex-wrap）
- 项目属性（flex-grow、flex-shrink、flex-basis、order）
- 常见布局模式（两栏、三栏、等高）
- 实践：用 Flex 实现圣杯布局、导航栏、卡片布局

### Day 4: Grid 布局精通

- 网格容器与网格项
- grid-template、gap、grid-area
- 自动填充与响应式（auto-fit、auto-fill、minmax）
- 实践：实现 Pinterest 瀑布流、仪表盘布局

### Day 5: CSS 进阶技巧

- BFC（Block Formatting Context）原理与应用
- 清除浮动的多种方法
- CSS 变量（自定义属性）
- calc()、clamp()、min()、max()
- 实践：使用 CSS 变量实现主题切换

### Day 6: Tailwind CSS 基础 ⭐

- Tailwind 核心理念（Utility-First）
- 安装配置与 VSCode 插件
- 常用工具类（spacing、colors、typography）
- 响应式前缀（sm、md、lg、xl、2xl）
- 伪类与伪元素（hover、focus、group）
- 实践：用 Tailwind 重构之前的布局

### Day 7: Tailwind CSS 进阶 ⭐

- 自定义配置（theme、colors、spacing）
- 暗黑模式实现
- 组件抽取（@apply、组件化）
- 插件系统（forms、typography、aspect-ratio）
- 性能优化（PurgeCSS、JIT 模式）
- 实践：实现完整的响应式个人网站（含暗黑模式）
